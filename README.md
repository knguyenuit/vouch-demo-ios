# Vouch-iOS-Demo

 ### Requirements ###
1. [Cocoapods](https://guides.cocoapods.org/using/getting-started.html#getting-started)
2. Xcode 13.0

### Installation ###
1. Clone project
2. cd path_to_project_folder
3. pod install
4. open VouchDemo.xcworkspace
5. Build & run.

### Architecture ###
1. MVVM design pattern.
2. RxSwift
